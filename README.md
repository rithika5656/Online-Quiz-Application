# Online Quiz Application

A simple MCQ quiz application with automatic score calculation.

## Features

- Multiple Choice Questions (MCQ)
- Auto score calculation
- Clean and simple UI
- Try again option

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open `index.html` in a web browser
2. Answer all the questions
3. Click "Submit Quiz" to see your score
4. Click "Try Again" to restart the quiz
